The two keys that need to be generated are done so using some advanced mathematics. The first thing to do is multiply our two prime numbers together, p and q to give us the value n. To create the other half of the public key, we need a specific value e. This is usually set to be 65,537 for efficiency.

Our private key is generated with a slightly different process. We need to find the Carmichael's Totient of n (λ(n)). This is equal to the lcm((p-1)(q-1)). We then calculate the decryption key d as being 1/e mod λ(n), which is using the inverse modulo operation.
